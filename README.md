# Ben Siverly - Personal Website

Personal website serving as professional home base, portfolio, and host for free tools.

## Tech Stack

- Clean HTML5, CSS3, minimal JavaScript
- No frameworks or build process
- Mobile-first responsive design
- Static hosting ready (GitHub Pages, Cloudflare Pages, Netlify)

## Local Development

Open `index.html` in a browser, or use a local server:

```bash
# Python
python -m http.server 8000

# Node.js (npx)
npx serve
```

## Deployment

Deploy to any static hosting platform by uploading the files or connecting the repository.
